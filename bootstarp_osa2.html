HTML-sivun perusrakenne

HTML koostuu tageista, jotka määrittelevät sivun rakenteen.

<html>

Määrittelee koko HTML-dokumentin alun ja lopun.

<head>

Sisältää sivun metatiedot kuten otsikon ja linkitykset.

<body>

Sivun näkyvä sisältö tulee tänne.

<p>

Kappaletekstiä varten.

<img>

Kuvan lisääminen sivulle.

<a>

Linkkien luomiseen.

<br>

Rivinvaihto ilman uutta kappaletta.

<h1>

Suurin otsikkotaso.

CSS-muotoilu

CSS:llä määritellään sivun ulkoasu. Esimerkiksi värit, fontit ja asettelu. 

body {
    background-color: lightgray;
}

h1 {
    color: blue;
}

CSS voidaan lisätä:
- suoraan HTML-tiedostoon (<style>)
- erilliseen .css-tiedostoon

Bootstrap

Bootstrap on valmis CSS-kirjasto, joka helpottaa sivujen tekemistä. Se sisältää valmiita komponentteja kuten:

  • Navbar (ylävalikko)
  • Card (kortti)
  • Grid-järjestelmä (asettelu)

Bootstrap lisätään sivulle linkittämällä sen CSS-tiedosto <head>-osioon.
